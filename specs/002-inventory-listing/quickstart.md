# Quickstart: Inventory Page Tests

**Feature**: 002-inventory-listing
**Target**: https://www.saucedemo.com/inventory.html
**Depends on**: `001-saucedemo-auth` (`.auth/standard-user.json` must exist)

## Prerequisites

```bash
npm install
npx playwright install   # installs Chromium, Firefox, WebKit
```

`.auth/standard-user.json` must exist. It is generated by running any session
test or manually via:

```bash
npx playwright test tests/e2e/session.spec.ts --project=chromium
```

## Environment Setup

`.env` at repository root (created by feature 001 setup):

```bash
SAUCE_USERNAME=standard_user
SAUCE_PASSWORD=secret_sauce
```

## Running Inventory Tests

```bash
# Run only inventory tests
npx playwright test tests/e2e/inventory.spec.ts

# Run on a single browser
npx playwright test tests/e2e/inventory.spec.ts --project=chromium

# Run all 3 browser projects
npx playwright test tests/e2e/inventory.spec.ts --project=chromium --project=firefox --project=webkit

# Headed mode (useful for debugging sort behaviour)
npx playwright test tests/e2e/inventory.spec.ts --headed --project=chromium

# Run all e2e tests (001 + 002)
npx playwright test tests/e2e/
```

## Troubleshooting

| Symptom | Likely cause | Fix |
|---------|-------------|-----|
| `Cannot find .auth/standard-user.json` | 001 global-setup never ran | Run any session test or re-run global setup |
| Sort test fails intermittently | Race condition — sort applied before products rendered | Ensure `sortBy()` awaits DOM update after selecting option |
| `getCartBadgeCount()` returns null unexpectedly | Cart state leaked from prior test | Check that storageState has empty cart; delete `.auth/` and regenerate |
| `isOnDetailPage()` returns false | Navigation didn't complete | Add explicit `waitForURL('/inventory-item.html')` in `clickProductByName()` |
| Price sort fails with equal prices | Tie-breaking assumption | Spec says relative order among equal prices is unspecified; test should only assert sorted reference array |
